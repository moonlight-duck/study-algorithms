## 정리
- 입출력, 사칙연산
  - 문제에서 원하는 출력을 정확히 파악하는 것이 필요
- 조건문, 반복문
  - 예제와 별개로 여러 케이스를 먼저 생각해보는 것이 필요
- 집합, 맵
  - 자바는 HashMap으로 수를 찾을 때 가장 빠르다
- 브루트 포스
  - 모든 수를 검사해도 시간이 충분할지 확인해볼 것
