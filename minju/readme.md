## 실행 방법

### 1. 기본 실행

```bash
# 백준 폴더로 이동 후 실행
pnpm ts-node 1000.ts
```

### 2. 입력과 함께 실행

```bash
# 파이프를 통한 입력
echo "3 5" | pnpm ts-node 1000.ts

# 여러 줄 입력
printf "3 5\n10 20\n" | pnpm ts-node 1000.ts
```

### 3. 직접 입력

```bash
# 터미널에서 직접 입력 (Ctrl+D로 종료)
pnpm ts-node 1000.ts
```

## 개발 환경

- **언어**: TypeScript
- **런타임**: Node.js
- **패키지 매니저**: pnpm
- **모듈 시스템**: ES Modules

## 스크립트

- `pnpm ts-node [파일명]` - TypeScript 파일 실행
- `pnpm lint` - 코드 린팅
- `pnpm lint:fix` - 자동 린팅 수정
- `pnpm format` - 코드 포맷팅
